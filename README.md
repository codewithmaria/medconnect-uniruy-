# MedConnect - Gestão de Atendimento Médico 
**Projeto PBL - Engenharia de Software | UniRuy Wyden**

O MedConnect é uma plataforma para automação de clínicas, focada em agendamento inteligente e prontuários digitais. Este repositório contém a documentação completa, modelagem de dados e o protótipo funcional.

## Tecnologias e Metodologias
- **Metodologia:** Ágil (Scrum e XP)
- **Protótipo:** HTML5, CSS3, JavaScript
- **Design:** Figma (Alta Fidelidade)
- **Banco de Dados:** PostgreSQL (DDL incluído)
- **Documentação:** UML 2.0 (Casos de Uso) e DFD

## Especificação de Requisitos

### Requisitos Funcionais (RF)
| ID | Nome | Descrição |
|----|------|-----------|
| RF01 | Gestão de Usuários | Cadastro de pacientes e médicos com validação de CPF/CRM. |
| RF02 | Agendamento Online | Sistema de reserva de horários com verificação de conflitos. |
| RF03 | Prontuário Digital | Registro de anamnese e prescrições por médicos autenticados. |
| RF04 | Painel Administrativo | Geração de métricas de atendimento e produtividade. |

### Requisitos Não Funcionais (RNF)
| ID | Nome | Descrição |
|----|------|-----------|
| RNF01 | Segurança (LGPD) | Criptografia de ponta a ponta para dados sensíveis. |
| RNF02 | Desempenho | Tempo de resposta de busca inferior a 2 segundos. |
| RNF03 | Responsividade | Interface adaptável para Desktop e Mobile. |
